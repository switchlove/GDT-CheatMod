This is a mod for Game Dev Tycoon! It enables users to use cheats through the main in-game menu!

Using this mod is a safer method then changing the save game directly with an editor.

How to activate CheatMod

  1. Startup the game and open the main menu.
  2. In the main menu open up the Mods screen.
  3. Select CheatMod and reboot Game Dev Tycoon.

Now when you press the left mouse button a new option called "CheatMode..." is added.
